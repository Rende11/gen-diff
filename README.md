## Install (global)
Avalible in npm
```
npm install -g gen-diff
```

## Usage
gen-diff [--format] filepath1 filepath2

[example](https://asciinema.org/a/du0fqdmfa9tn52styav1ywenk)

### Supported input files formats
* JSON
* YAML
* INI

### Avalible output formats
* pretty (default)
* plain text
* json
